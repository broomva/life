# life

Agent OS monorepo containing the core primitives that power the BroomVA agent infrastructure.

## Sub-crates

| Crate | Description |
|-------|-------------|
| **arcan** | Core runtime — event loop, capability system, agent lifecycle |
| **lago** | Persistence — durable state, memory, and storage layer |
| **aiOS** | Cognitive layer — LLM integration, reasoning, prompt orchestration |
| **autonomic** | Homeostasis — self-regulation, health checks, adaptive control |
| **spaces** | Networking — inter-agent communication, pub/sub, mesh topology |
| **praxis** | Tool execution — skill dispatch, sandboxed function calling |
| **vigil** | Observability — tracing, metrics, diagnostics |
