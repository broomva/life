# Roadmap

## Phase 1: Core Hardening (Now)

1. Deterministic replay harness and equivalence tests.
2. Crash-recovery failure-injection tests.
3. Extended tracing spans across kernel/runtime/tool boundaries.
4. Metrics endpoint and SLO baseline.

## Phase 2: Security and Isolation

1. Strong sandbox backend abstraction and implementation.
2. Egress policy hardening and secret mount policy.
3. Approval gate UX and operational controls.

## Phase 3: Distributed Control Plane

1. Session queueing/backpressure.
2. Multi-agent coordination primitives.
3. Multi-node event/log storage strategy.

## Phase 4: Release and Distribution

1. Artifact packaging strategy (containers/binaries).
2. Signed builds, SBOM, provenance attestations.
3. Ops runbooks, rollback procedure, and release cadence.
