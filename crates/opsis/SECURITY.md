# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public GitHub issue
2. Email **security@broomva.tech** with:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
3. You will receive acknowledgment within 48 hours
4. We will work with you to understand and address the issue

## Scope

This policy covers the Opsis frontend (`@opsis/web`, `@opsis/core`). For vulnerabilities in the Rust backend (`opsisd`), report to the [Life Agent OS](https://github.com/broomva/life) repository.
