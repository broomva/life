# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-06

### Added

- CesiumJS 3D globe with Google Photorealistic 3D Tiles
- SSE streaming from opsisd backend (earthquakes, weather)
- Tactical HUD overlay with liquid glass panels
- Dark space background with cyan accent glow
- Monorepo structure with Turborepo (apps/web + packages/opsis-core)
- Architecture documentation
- CI/CD with GitHub Actions
- Vercel deployment for frontend
- Railway deployment for opsisd backend
